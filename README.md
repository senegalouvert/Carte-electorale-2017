# Carte Électorale 2017
## Source
Carte électorale publiée sur le site la [Commission Électorale National Autonome (CENA)](https://www.cena.sn) du Sénégal

## Données
* Départements
* Collectivités
* Lieu de vote
* Numéro Bureau de votes
* Nombre électeur

## Format CSV
Les données sont au format csv
